# React Basic Topics Project

This small React project demonstrates 5 basic concepts:
1. State with useState (Counter)
2. Side effects & data fetching with useEffect (FetchUsers)
3. Forms & controlled inputs (FormDemo)
4. Context API (ContextDemo)
5. Routing with react-router-dom (Home, About, Profile)

How to run:
1. npm install
2. npm start

Each component file name shows the concept covered.
